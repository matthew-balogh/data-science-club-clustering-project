Overview of
Data Science Club Project
Clustering Data and Visualizing in 2D

Brief Objective

Group similar data points together and then visualize where and how those groups separate from each other and explore what characterizes them.

Topics covered

  • Clustering two-dimensional data with K-means
  • Clustering high-dimensional data, while visualizing it in 2D using Self-Organizing Maps
  • Touch up on how Self-Organizing Maps relate to neural networks

Expectations

Our analysis will be accompanied by visualizations and we expect that we can display even high-dimensional datasets in two dimensions that is interpretable for the human eye. Also, by visiting the listed topics, we expect that by the end of the project we will have both theoretical knowledge and practical experience in the field of Representation Learning.
